üîóIntegration Challenge: Behavior Change Event Design

30 Minutes

Combine your strategic event planning expertise with behavior change psychology to design a transformational community experience. This advanced integration challenge requires you to architect events that don't just inform, but actually motivate sustainable behavior change.

üéØ Integration Challenge
  • Primary Goal: Fuse event management skills with behavior change science
  • Deliverable: Complete behavior change event blueprint
  • Skills Integration: Event Planning + Motivational Interviewing + Behavior Psychology
Challenge Requirements & Deliverables
  • Behavior Change Event Blueprint: Design a 2-hour Club Stride community workshop that uses motivational interviewing principles to guide participants through a specific health behavior change journey
  • Psychological Flow Design: Create a detailed agenda showing how you'll use reflective listening, open-ended questions, and behavior change stages throughout the event timeline
  • Interactive Activity Framework: Develop 3 specific activities that apply motivational interviewing techniques in group settings (role-play scenarios, reflection exercises, peer conversations)
  • Resistance Strategy Plan: Anticipate and plan for common participant resistance or ambivalence, with specific motivational interviewing responses and techniques
  • Follow-up Behavior Support System: Design a 30-day post-event engagement plan that continues to support behavior change using motivational interviewing principles
  • Integration Reflection Essay: Write a 500-word analysis of how event planning logistics must be adapted when the goal shifts from information delivery to behavior transformation
